# Gulp Setup
## Getting Started

Open shell in your directory, and run that command:
```shell
git clone https://github.com/byCodeXp/gulp_setup.git
```
Go to the folder:
```shell
cd gulp_setup
```
Then:
```shell
npm install
```

## Requirements
- [Node.js](https://nodejs.org/en/)
- [Gulp.js](https://gulpjs.com)
